#快捷键大全
## Vscode
+ 全局命令面板：ctrl+shift + p
+ 查找/替换：ctrl + f/h
+ 全局查找：ctrl+shift + F
+ 注释：ctrl + /
+ 代码格式化：shift+alt + f

+ 多编辑器：ctrl + \ 
+ 打开/关闭多窗口：ctrl+shift + n/w
+ 移动到行首/行尾：home/end
+ 移动到文件开头/结尾：ctrl + home/end
+ 移动行： alt + 方向键上/下

+ 字体大小：ctrl + +/-
+ 选中一行：3击

## 搜狗
+ 简繁体切换：ctrl+shift + f
  
## Linux
+ 截屏：shift + PrtSc

## Windows
+ 打开cmd：shift+右键 ->(powershell) start cmd
  
## python (vscode)
+ 运行：终端 python xx.py
